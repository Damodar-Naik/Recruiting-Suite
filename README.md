This is a mockup for a Recruitment Suite

Stack :
- Nextjs, for frontend.
- NodeJs & express, for backend.
- Gemini, for LLM.
- AFFINDA, for Resume Parsing.
- postgres, for DB.

---------------------------------------------------------------------------
steps to setup the project loacally:

- clone the repository
- npm i
- npm run dev

env variables :
- NEXT_PUBLIC_BACKEND_URL
